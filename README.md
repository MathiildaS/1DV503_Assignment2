# Book Store Application
Developed as part of a database course at Linnaeus University.

## Installation
**1. Clone the repository**
```bash
git clone https://github.com/MathiildaS/1DV503_Assignment2.git
```

**2. Navigate to folder and install dependencies**
```bash
npm install
```

**3. Create a .env file in the root folder and provide your database connection details**
```See .env.example
```

**4. Start the application in development mode**
```bash
npm run dev

Server running at http://localhost:3000
```
